Fetal Health Prediction using Hybrid ML and Python

The Challenge: To accurately classify fetal health status from complex Cardiotocography (CTG) data to assist healthcare professionals in making timely decisions.

The Solution: Built a hybrid machine learning model combining the strengths of SVM and XGBoost. The model achieved an impressive **97.60% accuracy** in classifying fetal health into normal, suspect, and pathological states. We also used SHAP (SHapley Additive exPlanations) to provide model interpretability, which is critical for trust and adoption in healthcare.

Technologies Used: Python, Scikit-learn, XGBoost, SHAP, Data Analysis.

Currently Undergoing Project

AI Handwriting Generation System ✍️

The Challenge: To create a system that can learn a user's unique handwriting style from a single image and generate new content in that style from typed text.

The Solution: I am developing a full-stack AI system using Python and TensorFlow. The core of the system is a Conditional Variational Autoencoder (CVAE) that encodes handwriting style into a latent vector. A text encoder then guides the model to generate the desired text, which can be exported as a high-quality PDF or image file. The front-end, built with Streamlit, provides a simple interface for users to upload samples and generate text.

Technologies Used: Python, TensorFlow, Streamlit, HTML/CSS.
